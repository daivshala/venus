Venus UI

Ingresse UI FrameworK

Grid

Create mult-device layouts quickly and easily with the deafult twelve column, making for a 940px wide container with responsive features enabled.

Basic

For a simple two column layout, create a .row and add the appropriate number of columns.

8
4
<div class="venus row">
    <div class="venus column eight">8</div>
    <div class="venus column four">4</div>
</div>

The above code sets all columns in mobile mode in standard full width. To control the columns in Mobile mode, you must use the following code.

8
4
<div class="venus row">
    <div class="venus column eight mob-three">8</div>
    <div class="venus column four mob-one">4</div>
</div>

The mobile grid suport four columns: .mob-one, .mob-two, .mob-three and .mob-four.

Offsetting columns

Move columns to the right, using .offset-* classes. Each class increases the left margin of column by a whole column. For example .offset-4 moves column .column .four over four columns.

4
<div class="venus row">
    <div class="venus column four offset-four">4</div>
</div>
Centralizing columns

To center columns is very easy, just add the tag .centered and be happy.

5
<div class="venus row">
    <div class="venus column five centered">5</div>
</div>
Adaptive grids

By default the tag container, keeps a fixed grid of 940px. But it is possible to expand this grid, for use on larger monitors as well, and if you want to build a specific layout for tablets.

Adding the tag .transmute-grid, the container will expand according to the size of the monitors used. Enabling the grid to behave more columns, going from 12 to 20 columns.

<div class="venus container transmute-grid">
    <div class="venus row">
        <div class="venus twelve column"></div>
    </div>
</div>

To use the grid of 8 columns specific to tablets, just add the tag to the container .transmute-grid-mob

<div class="venus container transmute-grid-mob">
    <div class="venus row">
        <div class="venus twelve column"></div>
    </div>
</div>
